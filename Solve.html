The Palindrome Challenge

What is a Palindrome?

A palindrome is a word, phrase, number, or sequence of characters that reads the same backward as forward.

Classic examples include words like "racecar", "level", and "madam". But palindromes can also be entire phrases, like the title of this project: "Taco Cat"!

The Exercise

This coding challenge demonstrates how to detect palindromes using JavaScript. The application:

  • Takes user input from a text field
  • Reverses the string character-by-character using a for loop
  • Compares the original string to its reversed version
  • Displays the reversed string in an output field
  • Provides visual feedback using SweetAlert2 modals and confetti celebration
